# Contributing

Thanks for your interest.
External contributions are currently accepted only on a limited, maintainer-reviewed basis.
Issue reports are welcome.

## Pull Requests

- Open an issue first for non-trivial changes so scope can be confirmed.
- Pull requests may be closed if they are out of scope for the current roadmap.
